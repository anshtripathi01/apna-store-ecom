## Installation & Run Locally

Clone the project

```bash
 git clone https://github.com/anshtripathi01/apna-store-ecom.git
```

Go to the project directory

```bash
  cd apna-store-ecom
```

Open project

```bash
  npm install
```

Start the server

```bash
  npm start
```

## Key features
- Authentication

- Filter products by
    1. Price 
    2. Category
    4. Rating

- Cart Management
- Wishlist Management
- Product Listing Page
